# 🔮 나의 전생 이야기

이름을 입력하면 AI가 당신의 전생을 알려주는 흥미유발형 스토리텔링 서비스입니다.

기원전 15만 년 전부터 1980년까지, 랜덤한 시대와 직업이 부여되고 OpenAI가 드라마틱한 전생 이야기를 만들어줍니다.

---

## 서비스 구성

```
deploy_test1/
├── server.js          # Express 백엔드 (OpenAI API 연동)
├── jobs.json          # 300개 직업 목록 (실제 + 가상)
├── public/
│   └── index.html     # 프론트엔드 UI (단일 파일)
├── .env               # OpenAI API 키 (Git 미포함)
├── .gitignore
└── package.json
```

| 구성 요소 | 설명 |
|-----------|------|
| **백엔드** | Node.js + Express. `/api/past-life` 엔드포인트에서 랜덤 시대·직업·사인을 조합하고 OpenAI로 스토리 생성 |
| **프론트엔드** | 반응형 단일 HTML. 별이 반짝이는 신비로운 UI, 로딩 애니메이션, 결과 카드 표시 |
| **직업 데이터** | `jobs.json`에 300개 직업 저장. 석기시대 동굴 벽화가부터 1920년대 밀주 제조업자까지 |
| **AI 스토리텔링** | GPT-4o-mini가 이름·시대·직업·사인을 엮어 300자 내외의 감성적이고 유머있는 이야기 생성 |

## 설치 및 실행

### 1. 클론

```bash
git clone git@github.com:magicpsy85-create/deploy_test1.git
cd deploy_test1
```

### 2. 의존성 설치

```bash
npm install
```

### 3. 환경 변수 설정

프로젝트 루트에 `.env` 파일을 생성하고 OpenAI API 키를 입력합니다.

```
openai_api_key=sk-proj-여기에_본인의_API키_입력
```

### 4. 서버 실행

```bash
npm start
```

브라우저에서 `http://localhost:3000` 접속

## 사용법

1. 메인 화면에서 **이름 입력**
2. **"전생 알아보기"** 버튼 클릭
3. AI가 전생 스토리를 생성하는 동안 로딩 애니메이션 표시
4. 결과 카드에 **시대, 직업, 사인, 스토리** 표시
5. **"다시 보기"** → 같은 이름으로 새로운 전생 결과 생성
6. **"공유하기"** → Twitter 공유 또는 텍스트 클립보드 복사

## 응용 예시

### 파티/모임 이벤트
참가자들이 각자 이름을 입력하고 전생 결과를 공유하며 대화 소재로 활용. 가장 재미있는 전생 이야기에 투표하는 미니 게임으로 확장 가능.

### SNS 바이럴 콘텐츠
공유하기 기능을 활용해 Twitter 등에 전생 결과를 올리고, "나도 해보기" 링크로 자연스러운 바이럴 유도.

### 커플/친구 궁합 확장
두 사람의 전생을 비교해서 "전생에서도 만났을까?" 같은 궁합 스토리를 만드는 기능으로 확장 가능.

### 교육 콘텐츠
역사 수업에서 특정 시대의 직업과 생활상을 흥미롭게 전달하는 보조 도구로 활용. 학생들이 자신의 전생 직업에 대해 추가 조사하는 과제로 연결.

### 카페/팝업스토어 체험 부스
QR코드로 접속 → 전생 결과를 영수증 프린터로 출력 → 방문 기념품으로 제공.
